# 고속도로위 팀 그라운드 룰 

### 회의 🤝

- 정기 회의 일정 : 매주 캡스톤 디자인 수업 시간 활용
- 추가 회의 일정(필요시) : 매주 일요일 22시
- 회의 내용 노션에 기록 (기록 담당은 매주 돌아가며 배정)

### 목표 기한 📝

- 마감기한 엄수하기 (최소 마감 하루 전 제출)
- 스타트 때 기획 및 디자인 완료, 여름 방학부터 그로쓰까지 개발 진행
- 스타트 기간동안 기획 및 디자인 완료 + 시뮬레이션 프로그램 공부
- 겨울 방학부터 개발 진행

### 역할 😎

- 지우 - AI, 백엔드
- 지원 - 프론트엔드, 시뮬레이션 환경 세팅
- 재령 - AI, 백엔드
